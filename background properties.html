CSS Background Properties Example

CSS Background Properties Example

background-color

This element has a light blue background color.

background-image

This element has a background image.

background-position

This element's background image is centered.

background-repeat

This element's background image does not repeat.

background-attachment

This element's background image is fixed, so it stays in place as you scroll.

background-size

This element's background image covers the entire area.
